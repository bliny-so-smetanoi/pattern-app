Language: JavaScript

Documentation
-------------

Get started
-----------

Required environment:\
node v16.13.0\
npm 8.1.0
### Install all dependencies:
### `npm install`
### Available scripts:
### `npm run dev`
Runs the application in development mode.
### `npm run start`
Runs the application without nodemon.\
Application - http://localhost:5000
